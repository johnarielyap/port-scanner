# PORT SCANNER

THIS TOOL SCANS OPEN PORT ON AN IP LIKE NMAP TOOL.



# How to use
```
git clone https://github.com/johnarielyap/port-scanner.git
```

```
cd port-scanner
```

```
python portscanner.py
```
